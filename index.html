Coin Visualizer

Made with ❤️ by Stanley and Thomas
Drawing on The Web 2022 - NYU

This project is a web app that displays the price of a cryptocurrency in USD, and generates a random circle proportionally sized to the price of the cryptocurrency.

hint: try typing in the name of a cryptocurrency to see how it works.

Coinbase API link: https://developers. coinbase.com/api/v2 #prices
